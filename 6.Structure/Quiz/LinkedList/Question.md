# 연결 리스트
### 문제 분류
* 자료구조, 연결 리스트 

### 문제 개요
양방향 연결 리스트를 만들어라. 주어진 스켈레톤 코드의 linkedList.h를 참고하여 linkedList.c의 함수를 완성하여 main.c의 코드가 원활히 출력되도록 하라. 이때 linkedList.c 외 다른 파일은 편집할 수 없으며 추가적으로 파일을 생성해선 안된다.


### 답안 작성
